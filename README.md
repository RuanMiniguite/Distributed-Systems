## Installation

<h4>Requirements</h4>
<ol>
  <li>Install Python3</li>
</ol>


```
Git Clone https://github.com/RuanMiniguite/Template-Readme.git
```
```
sudo Python3 simple_topology.py
server1 xterm
h1 xterm
h2 xterm
h3 xterm
```
```
python3 server_chat_udp.py <IP_Host>
```
```
python3 client_chat_udp.py <IP_Host>
```
